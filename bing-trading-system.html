💡 彬哥心法：控制壞牌損失，放大好牌獲利。以下六條，每一筆單都嚴格遵守。
1單筆最大虧損 ≤ 總資金 2%
你的部位大小，由停損決定，不是由這把感覺會贏決定。500萬資金 → 單筆最多輸10萬。買100元股票，停損設90元(賠10元) → 最多買10張。
📐 使用停損計算機算出你的最大張數
2❌ 禁止攤平 — 輸錢不加碼
攤平是賭氣，不是策略。加碼是順勢增加獲利，才是王道。拿到一對3輸了，不蓋牌反而All-in？把攤平從字典刪掉，換成移動停損後的加碼。
✅ 加碼條件：確認牌面走勢有利才推籌碼
3連三週期望值為負 → 強制縮部位50%
職業賭徒記錄每一把決策好壞。你要算：勝率、平均獲利%、平均虧損%、期望值。連三週為負？縮到50%部位。這是自覺，不是認輸。
📊 使用期望值追蹤器監控
4AI股設時間停損 — 好牌不漲就是爛牌
產業對，股價不一定對。進場後10個交易日沒創新高，或漲幅低於停損幅度，先出場觀望。等牌不來第三張，果斷蓋牌。
⏱ 進場後計數10個交易日
5移動停損 — 讓獲利奔跑
賺30% → 停損移到成本之上。賺50% → 停損移到獲利20%位置。用贏來的籌碼賭更大的行情。好牌一定要贏到大，但不是凹單。
📈 每漲一段就把停損往上拉
6輸少贏多 — 青山在，魚永遠是你的
輸少 = 停損 + 2%上限 + 不攤平 + 期望值監控 + 時間停損
贏多 = 移動停損讓獲利奔跑
高手與輸家差別：判斷錯誤時損失更小，判斷正確時獲利更大。

⚡ 盤中實戰八條心法
①
開盤30分鐘沒波動 → 當日放棄
若波動小、量比低於0.8，場外大戶沒下單，當日就是散戶行情。尾盤掛被動單撿便宜，絕不主動拉抬。
②
先漲後跌、跌破開盤價 → 騙線出貨
早盤誘多手法，拉高讓散戶追，再緩跌出貨。跌破時5分鐘K線爆量 → 主力倒貨，進場等於幫人接籌碼。
③
先漲後跌但守住開盤價 → 洗盤完畢等第二波
主力刻意壓回測試散戶耐受力，若縮量守穩，浮額已輕。等第二次帶量突破早盤高點才按下買入鍵。
④
先跌後漲、突破開盤價 → 破底翻，強勢吸籌
主力開盤下殺製造恐慌，趁機吃貨。突破時量能倍增 + 期貨價差收斂 → 果斷跟進。這是真攻擊。
⑤
先跌後漲卻沒突破開盤價 → 弱反彈不參與
反彈無力且分時圖呈鋸齒狀，上方賣壓沉重。若持倉，減碼或佈局避險單。
⑥
尾盤最後7分鐘進場（13:23後）
當沖客強制出場，賣壓最輕。若最後一筆佔全日量能5%以上 → 主力進駐，隔日開高即收割。
⑦
微型飆股篩選：換手3-8% + 量比>1.5 + 市值<5億
籌碼活絡但未過熱。務必檢查集保戶數是否下降（籌碼集中），否則容易踩地雷。⚠️ 注意流動性風險。
⑧
長期下行股先放生 → 不接掉下來的刀子
股價在年線下方且季線下彎，反彈都是逃命波。唯一例外：窒息量（<百日均量40%）才考慮試單。

每天收盤後，做這兩件事：
🔍 功課一：看前五大買賣券商
買方是凱基-台北？
賣方是元大-總公司？
若兩者成立 → 隔日開盤前30分鐘就是決戰點。
📉 功課二：對照大盤多空
大盤5日線 ? 10日線
